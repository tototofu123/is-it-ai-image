Project Description

Build a CNN model in a Jupyter notebook to classify images as either real or AI-generated.

Objectives
- Use a local Python environment.
- Ensure required modules are installed (TensorFlow, Keras, and dependencies).
- Import dataset from:
	https://huggingface.co/datasets/Hemg/AI-Generated-vs-Real-Images-Datasets
- Load and preprocess the dataset.
- Build and train a Sequential CNN with Keras layers.
- Save the model in deployment-friendly formats (`.h5`, `.keras`, and SavedModel).

End Goal
- Produce a model that can be integrated into a website where users upload photos and receive a prediction: AI-generated or real.
